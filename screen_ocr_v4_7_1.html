How to use Screen OCR Helper

This tool captures a region of a shared screen/tab, runs on-device OCR in your browser, and lets you copy or search the recognized text.

1) Start capture

  1. Click Start and choose a screen or (recommended) a Chrome Tab.
  2. Zoom the source content to 150–200% for best OCR.

2) Select the region

  • Drag the blue box to move. Drag the bottom-right dot to resize.
  • For very small text, use a larger Scale in Options.

3) Run OCR

  • Manual: click OCR (or Ctrl+Shift+S) to scan once.
  • Continuous: switch to Continuous and set the interval. OCR will re-run automatically.

4) Use the results

  • Query tab: a single-line, joined version of the recognized text.
  • OCR tab: raw multi-line OCR output.
  • History tab: last 10 queries for quick reuse.
  • Copy: copies the Query text (Ctrl+Shift+C).
  • Search: opens results in your selected engine (Ctrl+Shift+G).

Options (⚙)

  • Auto-crop: trims the region to just detected text (can improve accuracy).
  • Multi-block auto-crop: for lists/feeds, keeps multiple text blocks so lower items aren’t dropped.
  • Scale: upscales the crop before OCR (3–4 helps small UI fonts).
  • Threshold: Fixed / Otsu / Bradley / Sauvola. Bradley/Sauvola often help with dark UI + anti-aliasing.
  • Morphology: Open removes specks; Close fills broken strokes.
  • Only update when text changes: reduces flicker in Continuous mode.
  • Prefer best confidence: keeps the highest-confidence OCR result over recent frames.
  • Auto-open search after OCR: opens a new tab automatically after OCR completes (use cooldown if enabled).
  • Custom engine: set a URL template using {q} (example: https://kagi.com/search?q={q}).

Troubleshooting

  • If text is missing: increase Scale, disable Auto-crop or increase Padding, and try Bradley/Sauvola.
  • If OCR is noisy: enable Only update when text changes and/or Morphology Open.
  • If capture looks blurry: share a single tab, zoom in, and avoid capturing motion/animations.
